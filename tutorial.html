How to create a blog on GitHub

GitHub tutorial:https://docs.github.com/en/get-started/quickstart/creating-an-account-on-github





2.create a blog

login, open homepage click plus(create new) sign then click New repository



Create a repository called [your_account_name.github.io]
Setting public



3.Check website address





4.upload website file


Note that blog homepage is the index.html file, which is also the default start page.
